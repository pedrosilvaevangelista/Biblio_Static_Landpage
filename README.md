## 📝 Sobre o Projeto
Este projeto é a materialização de uma **evolução no design**. Ele serve como uma atualização robusta de um projeto antigo, demonstrando como habilidades técnicas e estéticas podem amadurecer para alcançar níveis profissionais de UI/UX.

O **Biblio** foi reconstruído do zero para transformar uma ideia simples em uma experiência digital premium, com foco em design editorial, harmonia cromática e interatividade fluida.

### 🚀 Evolução & Design
O objetivo principal deste repositório é mostrar o contraste entre o passado e o presente:
- **Antes**: Design estático básico.
- **Agora**: Uma landing page moderna com grid layouts complexos, tipografia refinada e micro-interações.
### 🤖 O Papel da IA no Desenvolvimento
Neste projeto, a Inteligência Artificial foi utilizada como uma ferramenta de co-criação e aceleração. Acredito que, no cenário atual, ignorar o potencial da IA é um retrocesso; utilizá-la estrategicamente é essencial para não ficar para trás. Contudo, o foco central deste repositório permanece no **aprendizado contínuo** e na compreensão profunda de cada linha de código e decisão de design.

### ✨ Características
- 🎨 **Design Moderno**: Paleta de cores sofisticada (Peach, Coral, Teal).
- 📱 **Totalmente Responsivo**: Otimizado para dispositivos móveis, tablets e desktops.
- 🚀 **Performance**: Carregamento rápido e animações suaves.
- 🛠️ **Arquitetura Limpa**: Desenvolvido com HTML5, CSS3 vanila e JavaScript puro.

---

## 🛠️ Tecnologias Utilizadas
- **HTML5**: Estrutura semântica.
- **CSS3**: Estilização moderna, Flexbox e Grid Layout.
- **JavaScript (ES6+)**: Slider interativo e navegação dinâmica.
- **Google Fonts**: Tipografia selecionada para legibilidade e estilo.
